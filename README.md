# 🌑 Study Planner Widget (Electron)

Um aplicativo desktop minimalista, estilo widget, desenvolvido com **Electron** para organizar sua rotina de estudos com precisão. O app conta com temas personalizáveis (Dark e Rosa Fofo), relatórios de desempenho e um sistema inteligente de "Efeito Cascata" para lidar com imprevistos.

## ✨ Funcionalidades

- **🎯 Gestão de Atividades:** Planeje sua agenda semanal com blocos de estudo e pausas.
- **🚀 Sistema de Check-in (Start):** O cronômetro só começa quando você clica em "Começar". Caso atrase, o app empurra o fim da tarefa automaticamente.
- **⏸️ Pausa Inteligente:** Pausou para ir ao banheiro? O tempo de pausa é somado ao final da tarefa e empurra as próximas da agenda para não encavalar nada.
- **📊 Dashboard de Desempenho:** Acompanhe suas horas líquidas de foco, tarefas concluídas e faltas através de um histórico detalhado.
- **🎨 Temas Customizáveis:** Escolha entre o modo **Escuro Profundo** ou o modo **Rosa Vibrante (Fofo)**.
- **🖼️ Widget Always-on-Top:** O app pode ser fixado sobre outras janelas para você nunca perder o foco no timer.
- **👤 Perfil Personalizado:** Edite seu nome e foto de exibição.

## 🛠️ Tecnologias Utilizadas

- **Electron:** Framework para criação de apps desktop com tecnologias web.
- **HTML5 & CSS3:** Estrutura e estilização avançada (Variáveis CSS, Flexbox, Animations).
- **JavaScript (Vanilla):** Lógica do timer, manipulação de DOM e persistência de dados.
- **LocalStorage:** Armazenamento local dos dados da agenda e histórico.
